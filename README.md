# template for React with Vite

このリポジトリはViteでReactを使用するための最小限のセットアップを提供します。あくまで個人用に作成したテンプレートですが、何かの役に立つかもしれないので公開リポジトリにしています。

> This repository provides the minimum setup required to use React with Vite. It is a template created for personal use, but I have made it a public repository in case it may be useful to someone else.

## 構成

- Vite
- React
- TypeScript
- Chakra UI
- React Wouter
- React Icons
- Biome
- ...and more!

## 補足

Viteの生成コマンドを使って生成したものはそのまま残していますので不要のファイルも混入しているかもしれません。必要に応じて削除してください。

VIteが生成した`README.md`は`README.vite.md`に移動しています。

> The files generated using the Vite generation command are left as is, so there may be some unnecessary files mixed in. Please delete them as necessary.
>
> The `README.md` generated by Vite has been moved to `README.vite.md`.
